# 隐私政策

**最后更新：** 2026年1月

## 概述

Clipper 是一个免费开源的剪贴板管理系统。本隐私政策说明 Clipper 如何处理您的数据。

## 数据收集

**Clipper 不收集任何用户信息。**

- 不收集个人数据
- 不收集使用分析或遥测数据
- 不进行任何形式的跟踪
- 不向外部服务器传输任何数据

## 数据存储

您使用 Clipper 保存的所有剪贴板数据都存储在**您自己的设备本地**或**您控制的服务器**上：

- 使用桌面应用程序时，数据存储在本地数据库中
- 使用服务器组件时，数据存储在您自托管的服务器上
- 数据永远不会发送给 Clipper 开发者或任何第三方

## 开源

Clipper 完全开源。您可以：

- 查看源代码以验证这些隐私声明
- 审计软件对您数据的具体操作
- 根据您的需求修改软件

源代码：[https://github.com/windoze/clipper](https://github.com/windoze/clipper)

## 第三方服务

Clipper 不集成任何收集用户数据的第三方服务。如果您选择在云服务提供商上部署 Clipper，该提供商的隐私政策将适用于基础设施，但 Clipper 本身不会与外部服务共享任何数据。

## 剪贴内容分享功能

如果您使用可选的剪贴内容分享功能（短链接），分享的内容可通过您生成的 URL 访问。这是一个由您控制的可选功能：

- 只有您明确选择分享的内容才可被访问
- 分享链接会过期（默认：24小时）
- 分享功能默认关闭

## 政策变更

本隐私政策的任何变更都将在源代码仓库中更新本文档时体现。

## 联系方式

如对本隐私政策有任何疑问，请在项目的 GitHub 仓库中提交 issue。
